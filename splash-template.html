To create splash screens:

  1. Set viewport size to the target dimensions (e.g., 1290x2796 for iPhone 17 Pro)
  2. Hide these instructions (add ?hide-instructions to URL)
  3. Take a screenshot
  4. Save as splash-[width]x[height].png

For favicon:

  1. Add ?show-favicon to URL
  2. Take a 32x32 screenshot
  3. Save as favicon.ico
